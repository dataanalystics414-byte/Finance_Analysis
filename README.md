# 📊 Finance Analytics Dashboard | Power BI

## 📌 Overview

Developed an interactive **Finance Analytics Dashboard** in **Power BI** to analyze financial transactions, customer behavior, taxes, fees, and business performance. The dashboard provides real-time insights through dynamic visualizations and filters, enabling stakeholders to make data-driven decisions.

---

## 🎯 Business Objectives

- Monitor overall financial performance and KPIs
- Analyze monthly transaction trends
- Compare successful vs failed transactions
- Evaluate customer segment and gender contribution
- Track state-wise financial performance
- Analyze transaction type profitability
- Measure Year-over-Year (YoY) growth

---

## 🛠 Tools & Skills

- Power BI
- Power Query
- DAX
- Data Modeling
- Data Visualization

---

## 📊 Dashboard Highlights

- KPI Cards (Transactions, Customers, Fees, Taxes)
- Monthly Trend Analysis
- YoY Performance Analysis
- State-wise Performance
- Customer & Gender Analysis
- Transaction Type Analysis
- Interactive Slicers (Year, Occupation, Category, Dynamic Measure)
